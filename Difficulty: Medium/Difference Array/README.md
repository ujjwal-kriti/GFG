<pre><strong><span style="font-size: 14pt;">Input: </span></strong><span style="font-size: 14pt;">arr[] = [4, 5, 7, 9], opr[][] = [[1, 3, 1], [2, 3, -2]]</span><strong><span style="font-size: 14pt;"><br>Output: </span></strong><span style="font-size: 14pt;">[4, 6, 6, 8]<br></span><strong><span style="font-size: 14pt;">Explaination:<br></span></strong><span style="font-size: 14pt;">Operation [1, 3, 1] : Add 1 to indices 1 to 3 → [4, 6, 8, 10] <br>Operation [2, 3, -2] : Substract 2 from indices 2 to 3 → [4, 6, 6, 8] <br>Final array is [4, 6, 6, 8].</span></pre>
